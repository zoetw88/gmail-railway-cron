# PWA-001 — Inbox Daily 可安裝與緊急推播

## 驗收條件

- 儀表板可由手機安裝到主畫面，具有獨立顯示模式與應用程式資訊。
- 已授權的檢視者可在儀表板開啟或關閉這台裝置的通知。
- 每次摘要寫入後，只推播新摘要中的緊急郵件；一般與重要郵件只留在儀表板。
- 通知只包含寄件信箱代號、標題、安全摘要與 Gmail 連結，不包含郵件全文。
- 同一摘要不會重複推送到同一訂閱；失效訂閱會移除。
- 儀表板提供一個「立即整理」按鈕，一次處理全部信箱，並有五分鐘防重複啟動。
- 「立即整理」與「緊急通知」位於一般內容流的統一工具區，不固定或黏附於畫面頂端。
- 四個信箱預設收合，摘要列顯示處理狀態與數量；展開後保留分類、AI 重點及 Gmail 連結。
- 主要操作具備清楚的載入、成功、停用、錯誤及鍵盤焦點狀態，手機觸控目標至少 44px。
- 390×844 首屏直接顯示「現在是否有急事」與第一封緊急或重要郵件；整理時間降為次要資訊。
- 緊急郵件為零時不出現紅色警示，只顯示高度不超過 48px 的安心訊息。
- 重要與一般郵件分開；一般郵件及來源信箱預設收合。
- 「立即整理」是唯一主要操作；通知控制是次要裝置設定，不與整理命令等權。
- 整理啟動後自動確認新摘要並更新頁面；若等待逾時，需提供清楚且可恢復的狀態。
- 訂閱 API 需經 ChatGPT 登入及允許清單驗證；推播金鑰只存於部署環境。
- 儀表板建置、來源契約測試與 Python 整理測試通過。
